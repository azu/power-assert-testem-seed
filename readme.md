# power-assert-testem-seed

Example project of [power-assert](https://github.com/twada/power-assert "power-assert") + [gulp-espower](https://github.com/twada/gulp-espower "gulp-espower").

## Installation

``` sh
npm install
npm install -g gulp
```

## Usage

``` sh
gulp test
```
